Operating Systems


Basics

Operating systems (OS) are what run your computer on a daily basis and make you able to use it. Operating systems can be flashed to a USB through apps. These then can be used to install an operating system to any computer.


Windows

There are many types of operating systems. Firstly the most used OS as of June 2021 72.98% of people were using Windows 10. This is the most popular OS becasue it is easy to use and taught at many schools. They are also most commonly sold with laptops and desktops. It also runs every app that most people need. If you do decide to build a PC windows cost about AUD$200 to buy. Windows isn't as customisable than Linux.
